  • LeetCode 7: Reverse Integer

    March 02, 2025 Medium
    1. Initialize res to store the reversed integer, remainder for the last digit, and isNegative flag to handle negative inputs.
    2. If x is negative, negate it and set isNegative to true.
    3. Enter a loop that continues as long as x > 0:
      • Check if appending the next digit will cause overflow by comparing res with Integer.MAX_VALUE / 10. If so, return 0.
      • Multiply res by 10 to shift the existing digits left.
      • Extract the last digit of x using x % 10 and store it in remainder.
      • Add remainder to res.
      • Divide x by 10 to remove the last digit.
    4. If isNegative is true, negate res.
    5. Return the final reversed integer res.
